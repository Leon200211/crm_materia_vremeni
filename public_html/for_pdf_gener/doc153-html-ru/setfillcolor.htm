SetFillColor

SetFillColor(int r [, int g, int b])

Версия

1.3

Описание

Задает цвет, используемый для всех операций, использующих заливку (прямоугольники с заливкой и цвет фона ячеек). Может быть выражет в RGB или в градациях серого. Этот метод может вызываться до создания первой страницы и его значение сохраняется при переходе от страницы к странице.

Параметры

r
Если g и b заданы - красная составляющая; если нет - градация серого. Значение от 0 до 255.
g
Зеленая составляющая (от 0 до 255).
b
Синяя составляющая (от 0 до 255).

SetDrawColor(), SetTextColor(), Rect(), Cell(), MultiCell().
