Error

Error(string msg)

Версия

1.0

Описание

Этот метод автоматически вызывается при возникновении фатальной ошибки; он просто выводит сообщение о прекращении выполнения. Наследующий класс может переопределить этот метод для изменения обработки ошибки, но при этом он обязательно должен останавливать работу скрипта, иначе результирующий документ будет неправильным.

Параметры

msg
Сообщение ошибки.
