SetDrawColor

SetDrawColor(int r [, int g, int b])

Версия

1.3

Описание

Задает цвет для всех операций, прорисовующих контуры (линии, прямоугольники и границы ячеек). Может задаваться в RGB или градациях серого. Этот метод может вызываться до создания первой страницы и его значение не изменяется при переходе от страницы к странице.

Параметры

r
Если g и b заданы - красная составляющая; если нет - градация серого. Значение от 0 до 255.
g
Зеленая составляющая (от 0 до 255).
b
Синяя составляющая (от 0 до 255).
