AddLink

int AddLink()

Версия

1.5

Описание

Создает новую внутреннюю ссылку и возвращает ее идентификатор. Внутренняя ссылка - область, при нажатии на которую можно перейти в другое место документа.
Идентификатор может указывать на Cell(), Write(), Image() или Link(). Цель задается с помощью SetLink().

См. также

Cell(), Write(), Image(), Link(), SetLink().
