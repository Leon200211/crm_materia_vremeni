Write

Write(float h, string txt [, mixed link])

Версия

1.5

Описание

Этот метод выводит текст от текущей позиции. При достижении правой границы (или встрече символа \n) строка обрывается и текст продолжает выводиться с левого края. После выполнения текущая позиция остается в конце текста.
Возможно выведение ссылки в тексте.

Параметры

h
высота строки.
txt
Строка для вывода.
link
URL или идентификатор, возвращаемый AddLink().

Пример

//начинаем с установки шрифта
$pdf->SetFont('Arial','',14);
$pdf->Write(5,'Visit ');
//теперь добавляем синее подчеркивание для ссылки
$pdf->SetTextColor(0,0,255);
$pdf->SetFont('','U');
$pdf->Write(5,'www.fpdf.org','http://www.fpdf.org');

См. также

SetFont(), SetTextColor(), AddLink(), MultiCell(), SetAutoPageBreak().
