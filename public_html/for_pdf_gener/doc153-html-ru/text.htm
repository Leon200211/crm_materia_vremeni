Text

Text(float x, float y, string txt)

Версия

1.0

Описание

Выводит символьную строку. Начало - слева от первого символа, на основной строке. Этот метод позволяет точно разместить строку на странице, но, обычно, проще использовать стандартные методы для вывода текста - Cell(), MultiCell() или Write().

Параметры

x
Абсцисса начала.
y
Ордината начала.
txt
Строка для вывода.

См. также

SetFont(), SetTextColor(), Cell(), MultiCell(), Write().
