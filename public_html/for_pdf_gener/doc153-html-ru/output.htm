Output

string Output([string name [, string dest]])

Версия

1.0

Описание

Отображает документ документ в: строку, локальный файл или в браузер. В последнем случае может использоваться plug-in (если есть) или может быть вызвано диалоговое окно для сохранения ("Save as").
Если нужно, то сначала будет вызван метод Close() для закрытия документа.

Параметры

name
Имя файла. Если этот параметр не указан, то документ будет выведен в браузер (адрес I) с именем doc.pdf.
dest
Пункт назначения куда выводить документ. Может принимать одно из следующих значений:
  • I: вывести документ в браузер. Если возможно, используется plug-in. Имя заданное в name используется, если выбирается опция "Save as" на ссылке, генерирующей PDF.
  • D: отправляет в браузер и вызывает закачку файла с именем, заданным в name.
  • F: сохраняет в локальный файл с именем, указанным в name.
  • S: возвращает документ в виде строки. name игнорируется.
Если параемтр не задан, а имя задано, то пункт назначения F. Если ни один параметр не задан, то пункт назначения I.

Замечание: для совместимости с более ранними версиями также принимается логическое значение (false для F и true для D).
