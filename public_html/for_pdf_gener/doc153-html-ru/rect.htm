Rect

Rect(float x, float y, float w, float h [, string style])

Версия

1.0

Описание

Выводит прямоугольник. Может быть прорисован только по контуру, только заливкой (без границ) или залитым с контуром.

Параметры

x
Абсцисса верхнего левого угла.
y
Ордината верхнего севого угла.
w
Ширина.
h
Высота.
style
Стиль отображения. Может быть:
  • D или пустая строка: только контуры. Значение по умолчанию.
  • F: заливка
  • DF или FD: контур и заливка.

См. также

SetLineWidth(), SetDrawColor(), SetFillColor().
