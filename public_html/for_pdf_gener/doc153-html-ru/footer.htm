Footer

Footer()

Версия

1.0

Описание

Этот метод используется для визуализации нижнего колонтитула страницы. Он автоматически вызывается методом AddPage() и Close() и его не обязательно вызывать непосредственно из приложения. Реализация этого метода в FPDF не определена, так что требуется специальная обработка, то нужно создать подкласс и переписать данный метод.

Пример

class PDF extends FPDF
{
function Footer()
{
    //Отступаем снизу 1.5 см
    $this->SetY(-15);
    //Выбираем шрифр Arial курсив разером 8
    $this->SetFont('Arial','I',8);
    //Выводим номер страницы по центру
    $this->Cell(0,10,'Page '.$this->PageNo(),0,0,'C');
}
}
