SetTextColor

SetTextColor(int r [, int g, int b])

Версия

1.3

Описание

Задает цвет текста. Может быть выражен в RGB или в градациях серого. Этот метод может быть вызван до создания первой страницы и заданное значние сохраняется при переходе от страницы к странице.

Параметры

r
Если g и b заданы - красная составляющая; если нет - градация серого. Значение от 0 до 255.
g
Зеленая составляющая (от 0 до 255).
b
Синяя составляющая (от 0 до 255).
