Link

Link(float x, float y, float w, float h, mixed link)

Версия

1.5

Описание

Выводит ссылку на прямоугольную область страницы. Ссылки на текст или изображение обычно выводятся с помощью Cell(), Write() или Image(), но этот метод может быть полезен в таких исключительных случаях как, например, задание ссылки на область внутри изображения.

Параметры

x
Абсцисса верхнего левого угла прямоугольника.
y
Ордината верхнего левого угла прямоугольника.
w
Ширина прямоугольника.
h
Высота прямоугольника.
link
URL или идентификатор, возвращаемый AddLink().

См. также

AddLink(), Cell(), Write(), Image().
