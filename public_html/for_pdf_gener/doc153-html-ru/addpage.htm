AddPage

AddPage([string orientation])

Версия

1.0

Описание

Добавляет новую страницу в документ. Если страница уже есть, то сначала вызывается метод Footer() для вывода нижнего колонтитула. Когда страница добавлена, текущая позиция устанавливается в верхний левый угол соответственно к левой и верхней границе, вызывается метод Header() для вывода заголовка.
Установленный шрифт автоматически возобновляется. Так что нет необходимости снова вызывать SetFont(), если нужно продолжить работу с тем же шрифтом. Это справедливо и для цветов и ширины линий.
Начало координат в левом верхнем углу, увеличение значений идет вниз.

Параметры

orientation
Ориентация страницы. Допустимые значения (регистронезависимы):
  • P или Книжная
  • L или Альбомная
Значение по умолчанию такое, как задано в конструкторе.
