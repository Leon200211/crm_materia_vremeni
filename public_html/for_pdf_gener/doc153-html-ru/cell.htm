Cell

Cell(float w [, float h [, string txt [, mixed border [, int ln [, string align [, int fill [, mixed link]]]]]]])

Версия

1.0

Описание

Выводит ячейку (прямоугольная область) с заданными границами, цветом фона и строкой. Верхний левый угол ячейки соответствует текущей позиции. Текст может быть выровнен. После вызова этого метода текущая позиция перемещается вправо или на следующую строку. Возможно задать ссылку на текст.
Если задан автоматический разрыв страниц и ячейка превышает установленный лимит, то разрыв страницы произойдет перед выводом ячейки.

Параметры

w
Ширина ячейки. Если задан 0, то ячейка будет расширена до правой границы.
h
Высота ячейки. По умолчанию: 0.
txt
Строка для вывода. По умолчанию: пустая строка.
border
Задает должна ли выводиться граница вокруг ячейки. Значение может быть задано одним из следующих чисел:
  • 0: без границ
  • 1: с границами
или же некоторыми или всеми следующими символами (в любой последовательности):
  • L: левая граница
  • T: верхняя граница
  • R: правая граница
  • B: нижняя граница
По умолчанию: 0.
ln
Указывает где должна будет находиться текущая позиция после вывода ячейки. Допустимые значения:
  • 0: справа
  • 1: вначале следующей строки
  • 2: под ячейкой
Заданное значение 1 эквивалентно заданию значения 0 последующего вызова метода Ln(). По умолчанию: 0.
align
Позволяет выравнивать текст. Допустимые значения:
  • L или пустая строка: выравнивание по левому краю (значение по умолчанию)
  • C: выравнивание по центру
  • R: выравнивание по правому краю
fill
Указывает будет ли произведена заливка фона ячейки (1) или фон будет прозрачным (0). По умолчанию: 0.
link
URL или идентификатор, возвращенный методом AddLink().

Пример

//Задаем шрифт
$pdf->SetFont('Arial','B',16);
//Двигаемся на 8 см вправо
$pdf->Cell(80);
//В ячейке 20х10 мм с разрывом строк выравнивание текста задаем по центру
$pdf->Cell(20,10,'Title',1,1,'C');
