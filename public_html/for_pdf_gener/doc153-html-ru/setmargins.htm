SetMargins

SetMargins(float left, float top [, float right])

Версия

1.0

Описание

Задает левую, верхнюю и правую границы. По умолчанию все они равны 1 см. Для изменения используйте этот метод.

Параметры

left
Левая граница.
top
Верхняя граница.
right
Правая граница. По умолчанию - левая.

См. также

SetLeftMargin(), SetTopMargin(), SetRightMargin(), SetAutoPageBreak().
