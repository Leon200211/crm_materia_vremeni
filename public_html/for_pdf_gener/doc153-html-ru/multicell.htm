MultiCell

MultiCell(float w, float h, string txt [, mixed border [, string align [, int fill]]])

Версия

1.3

Описание

Этот метод позволяет печатать текст с переносами строк. Строки могут переноситься автоматически (как только будет достигнута правая граница ячейки) или явно (по знаку \n). Выводится столько ячеек, сколько необходимо - одна под другой.
Текст может быть выровнян по какому-то краю, по центру или по ширине ячейки. Текст может быть заключен в рамку и залит фоном.

Параметры

w
Ширина ячейки. Если 0, ячейки расширяются до правого края страницы.
h
Высота ячеек.
txt
Выводимый текст.
border
Указывает должна ли быть граница вокруг блока ячеек. Значение может быть задано одним из следующих чисел:
  • 0: без границы
  • 1: граница
или строка, содержащая или несколько следующих символов (в любом порядке):
  • L: левая граница
  • T: верхняя граница
  • R: правая граница
  • B: нижняя граница
По умолчанию: 0.
align
Задает выравнивание текста. Допустимые значения:
  • L: выравнивание по левому краю
  • C: выравнивание по центру
  • R: выравнивание по правому краю
  • J: по ширине ячейки (значение по умолчанию)
fill
Указывает, заливается фон (1) или будет прозрачным (0). По умолчанию: 0.
