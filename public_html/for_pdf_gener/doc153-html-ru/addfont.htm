AddFont

AddFont(string family [, string style [, string file]])

Версия

1.5

Описание

Импортирует TrueType или Type1 шрифт и делает его доступным. Сначала необходимо сгенерировать файл определения шрифта с помощью утилиты makefont.php.
Файл определения (и сам файл шрифта во время внедрения) должен присутствовать в каталоге шрифтов. Если он не найден, будет выдана ошибка "Could not include font definition file".

Параметры

family
Семейство шрифта. Название может быть выбрано произвольно. Если это название стандартного семейства шрифтов, то оно перезапишет соответствующий шрифт.
style
Стиль шрифта. Допустимые значения (регистронезависимы):
  • пустая строка: обычный
  • B: жирный
  • I: курсив
  • BI или IB: жирный курсив
По умолчанию - обычный шрифт.
file
Файл определения шрифта.
По умолчанию название формируется из названия семейства и стиля шрифта, в нижнем регистре и без пробелов.

Пример

$pdf->AddFont('Comic','I');
или, что то же самое: 
$pdf->AddFont('Comic','I','comici.php');
