AliasNbPages

AliasNbPages([string alias])

Версия

1.4

Описание

Задает псевдоним общего числа страниц. Оно будет заменяться когда документ закрыт.

Параметры

alias
Псевдоним. Значение по умолчанию: {nb}.

Пример

class PDF extends FPDF
{
function Footer()
{
    //Отступаем 1,5 см снизу
    $this->SetY(-15);
    //Выбираем Arial курсив размером 8
    $this->SetFont('Arial','I',8);
    //Выводим текущий номер и общее количество страниц
    $this->Cell(0,10,'Page '.$this->PageNo().'/{nb}',0,0,'C');
}
}

$pdf=new PDF();
$pdf->AliasNbPages();
