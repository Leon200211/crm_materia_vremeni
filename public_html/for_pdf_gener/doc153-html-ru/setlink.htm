SetLink

SetLink(int link [, float y [, int page]])

Версия

1.5

Описание

Задает страницу и позицию, куда указывает ссылка.

Параметры

link
Идентификатор ссылки, полученный с помощью AddLink().
y
Ордината цели ссылки; -1 обозначает текущую позицию. По умолчанию 0 (верх страницы).
page
Номер страницы, на которую делается ссылка; -1 обозначает текущую позицию. Это значение по умолчанию.

См. также

AddLink().
