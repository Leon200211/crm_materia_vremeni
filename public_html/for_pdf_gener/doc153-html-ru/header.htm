Header

Header()

Версия

1.0

Описание

Этот метод используется для задания заголовка страницы. Он автоматически вызывается методом AddPage(), так что его не обязательно вызывать неосредственно из приложения. Реализация этого метода в FPDF пуста, поэтому если нужно задать специфическую обработку, нужно переписать этот метод.

Пример

class PDF extends FPDF
{
function Header()
{
    //выбираем Arial размера 15
    $this->SetFont('Arial','B',15);
    //Передвигаемся вправо
    $this->Cell(80);
    //Заголовок с границами
    $this->Cell(30,10,'Title',1,0,'C');
    //Разрыв строки
    $this->Ln(20);
}
}
